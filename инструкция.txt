# Полная последовательность команд для запуска проекта DefectVision

## Предварительные требования

Перед запуском убедитесь, что на вашем компьютере установлены следующие программы:

- **Docker Desktop** (для Windows) — скачать с https://www.docker.com/products/docker-desktop/
- **Python 3.11** или выше — скачать с https://www.python.org/downloads/
- **Node.js 18** или выше — скачать с https://nodejs.org/
- **Git** — скачать с https://git-scm.com/
- **Miniconda** (опционально, для управления Python окружением)

После установки Docker Desktop запустите его и дождитесь, пока индикатор в трее станет зеленым.

## Часть 2: Запуск инфраструктуры (PostgreSQL и MinIO)


### Запустите инфраструктуру:

```bash
docker-compose -f docker-compose.infrastructure.yml up -d
```

### Проверьте, что контейнеры запущены:

```bash
docker ps
```

Ожидаемый вывод: два контейнера `defectvision-postgres` и `defectvision-minio` в статусе `Up`

### Создайте бакет в MinIO:

Откройте браузер и перейдите по адресу http://localhost:9001

- **Логин:** `minioadmin`
- **Пароль:** `minioadmin`

Нажмите **"Create Bucket"**, введите имя `defectvision`, нажмите **"Create Bucket"**

### Проверьте PostgreSQL:

```bash
docker exec defectvision-postgres psql -U postgres -c "SELECT 1"
```

Ожидаемый вывод: `?column?` и `1`

## Часть 3: Настройка и запуск бэкенда

### Создайте виртуальное окружение и установите зависимости:

```bash
cd C:\Users\"User_name"\Desktop\Eye_of_God\backend\backend

# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения (Windows)
venv\Scripts\activate

# Обновление pip
python -m pip install --upgrade pip

# Установка зависимостей
pip install -r C:\Users\romka\Desktop\-\GPO\Eye_of_God\requirements.txt
```

### Создайте таблицы в базе данных:

```bash
# Создание таблиц через Python
python -c "
import asyncio
from sqlalchemy.ext.asyncio import create_async_engine
from sqlalchemy import text
from app.core.config import settings
from app.models.base import Base
from app.models.user import User
from app.models.project import Project
from app.models.dataset import Dataset, Image

async def init():
    engine = create_async_engine(settings.DATABASE_URL, echo=True)
    async with engine.begin() as conn:
        await conn.run_sync(Base.metadata.create_all)
    print('Tables created successfully!')
    await engine.dispose()

asyncio.run(init())
"
```

### Запустите бэкенд:

```bash
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**Оставьте этот терминал открытым!**

## Устранение возможных проблем

### Ошибка "port already in use"

```bash
# Найти процесс, занимающий порт
netstat -ano | findstr :8000

# Убить процесс (замените PID на реальный)
taskkill /PID 12345 /F
```

### Ошибка подключения к PostgreSQL

```bash
# Перезапустить контейнер
docker restart defectvision-postgres

# Проверить логи
docker logs defectvision-postgres
```

### Ошибка CORS

В файле `backend/backend/app/main.py` убедитесь, что CORS настроен правильно:

```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173", "http://127.0.0.1:5173"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```
